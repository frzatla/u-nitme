# U-NIT ME

An AI-powered course planning web app for Monash University students. Enter your degree details and get a personalized, semester-by-semester course roadmap вЂ” instantly.

Built for the **UniHack** hackathon.

---

## Features

- **Personalized course plans** вЂ” generated by a 1,303-line Python scheduling algorithm using DAG critical-path analysis
- **Prerequisite-aware scheduling** вЂ” units are sequenced correctly based on prerequisite chains
- **Interactive plan editor** вЂ” drag and drop units between semesters
- **Unit details** вЂ” view offerings, assessments, and requisites pulled from the Monash Handbook
- **Student reviews** вЂ” see real r/Monash Reddit discussions for any unit
- **Save & manage plans** вЂ” authenticated dashboard to store and revisit plans
- **Elective search** вЂ” search 5,000+ Monash units by keyword, code, or level to fill free elective slots
- **AI unit advisor** вЂ” chat with "Ask ChatJKW" (powered by Groq + Llama 3.3) to get personalised unit recommendations

## How It Works

1. Sign in via Clerk
2. Choose your **course**, **area of study**, and optional **major/minor**
3. The server runs `algo1.py` вЂ” a 1,303-line Python scheduler that maps out your full degree
4. Your plan is saved to Supabase and displayed as an interactive semester grid
5. Click any **Free Elective** slot to search 5,000+ units or ask the AI chatbot for suggestions
6. Click any unit to view handbook details and Reddit reviews
7. Drag and drop to reorder, then save your plan

## Getting Started

### Prerequisites

- Node.js 18+
- Python 3 (for the scheduling algorithm)
- A [Clerk](https://clerk.com) account
- A [Supabase](https://supabase.com) project
- An [Elastic Cloud](https://cloud.elastic.co) deployment
- A [Groq](https://console.groq.com) API key (free)

### Installation

```bash
git clone https://github.com/frzatla/u-nitme
cd u-nitme/my-app
npm install
```

### Environment Variables

Create `my-app/.env.local`:

```
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_DEFAULT_KEY=
ELASTICSEARCH_CLOUD_ID=
ELASTICSEARCH_API_KEY=
GROQ_API_KEY=
```

### Index Units (one-time setup)

After setting your Elasticsearch env vars, index the unit database:

```bash
cd my-app
node scripts/index-units.js
```

This bulk-indexes 5,237 Monash units into your Elastic Cloud cluster. Only needs to be run once.

### Run

```bash
cd my-app
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
