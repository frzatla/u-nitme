# U-NIT ME

An AI-powered course planning web app for Monash University students. Enter your degree details and get a personalized, semester-by-semester course roadmap — instantly.

Built for the **UniHack** hackathon.

---

## Features

- **Personalized course plans** — generated by a custom Python scheduling algorithm using DAG critical-path analysis
- **Prerequisite-aware scheduling** — units are sequenced correctly based on prerequisite chains
- **Interactive plan editor** — drag and drop units between semesters
- **Unit details** — view offerings, assessments, and requisites pulled from the Monash Handbook
- **Student reviews** — see real r/Monash Reddit discussions for any unit
- **Save & manage plans** — authenticated dashboard to store and revisit plans

| Database | Supabase (PostgreSQL) |

| Algorithm | Python (`algo1.py`) |

---

## How It Works

1. Sign in via Clerk
2. Choose your **course**, **area of study**, and optional **major/minor**
3. The server runs `algo1.py` — a 1,300-line Python scheduler that maps out your full degree
4. Your plan is saved to Supabase and displayed as an interactive semester grid
5. Click any unit to view handbook details and Reddit reviews
6. Drag and drop to reorder, then save your plan

## Getting Started

### Prerequisites

- Node.js 18+
- Python 3 (for the scheduling algorithm)
- A [Clerk](https://clerk.com) account
- A [Supabase](https://supabase.com) project

### Installation

```bash
git clone https://github.com/frzatla/u-nitme
cd u-nitme/my-app
npm install
```

### Environment Variables

Create `my-app/.env.local`:

```
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_DEFAULT_KEY=
```

### Run

```bash
cd my-app
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
